# User Management System

A PHP OOP User Management System built as a University Assignment.

## Versions

- <b>Version 1:</b> Basic PHP OOP structure (Abstract class, Interface, Trait)
- <b>Version 2:</b> MySQL database integration with PDO
- <b>Version 3:</b> Web interface with PHP sessions (login, logout, register)

## Technologies Used

- PHP
- MySQL
- XAMPP
- HTML/CSS

## How to Run

1. Clone the repository
2. Place the project in <code>htdocs</code> folder in XAMPP
3. Start Apache and MySQL in XAMPP
4. Open browser and go to <code>http://localhost/user-management/</code>
